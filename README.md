# Otopark Yönetim Sistemi

Bu proje Bilgisayar Bilimine Giriş dersi kapsamında Java dili kullanılarak geliştirilmiştir.

## Proje Amacı
Otoparkta bulunan araçların giriş ve çıkış işlemlerini takip etmek.

## Özellikler
- Araç girişi
- Araç çıkışı
- Otoparktaki araçları listeleme
- Toplam araç sayısını gösterme

## Kullanılan Teknolojiler
- Java
- Eclipse IDE

## Kullanılan Konular
- Değişkenler
- Koşul ifadeleri
- Döngüler
- Metotlar
- ArrayList
